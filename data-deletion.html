Data Deletion Instructions

This page provides a public URL for data deletion requests related to the website and the published contact channels.

Last updated: May 28, 2026

Current status

This V1 site does not have accounts, login, its own database, forms, or backend. Because of that, the site does not store visitor profile data.

How to request deletion

If you have contacted Diez42 through Facebook, Instagram, WhatsApp, or another external channel and want to request deletion of data Diez42 may hold, contact Diez42 using the published channels on the homepage.

Include the channel you used, the approximate date of contact, and the information you want deleted. Do not send unnecessary sensitive data.

Data controlled by external services

If you want to delete data from an external account, such as Facebook, Instagram, or WhatsApp, use the account and privacy tools provided by that service. Diez42 does not control data stored directly by external services.

Future changes

If a future version adds login, forms, donations, or an external integration that stores data, this page should be updated with more specific instructions or a deletion callback if appropriate.
